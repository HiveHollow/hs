Stride /or Pitch as they sometimes call it,
     is the number of bytes from one row of pixels
     in memory to the next row of pixels in memory.

     Image format defines a minimum stride for an image.
     For uncompressed RGB formats, the minimum stride is
     always the image width in bytes rounded up to the
     nearest DWORD(32bit unsigned integer).
     Here's a formula for calculating stride:

      = ((((biWidth * biBitCount) + 31) & ~31) >> 3)
