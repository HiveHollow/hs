    Within a function, 'void' is another type.
    It does not return a value after it executes.
    When it's used for a function's parameter list,
    it specifies that the function takes no parameters.
    When used in the declaration of a pointer (a void *pointer),
    it specifies that the pointer is universal.

    You cannot declare a variable as void.
    A void *pointer can't be dereferenced unless it's cast to another type. 

    Here is a detailed interpretation...
    The Unary indirection operator (*) dereferences a pointer; that is,
    it converts a pointer value to an l-value.

    The operand of the indirection operator must be a pointer to a
    type. The result of the indirection expression is the same type
    as the pointer's type. In this context, you can distinguish
    the use of the (*) operator as Unary, versus its meaning
    through binary operation, multiplication.

    If the operand points to a function, the result is a function designator.
     func(a,b); // func is the designator
    If it points to a storage location, the result is an l-value designating
    the storage location (and I give an example with lvalue\rvalue later on).
    The indirection operator may be used cumulatively to dereference (ptr to ptr).

     int main() {
        int n = 5;
        int *pn = &n;
        int **ppn = &pn;

        cout  << "Value of n:\n"
              << "direct value: " << n << endl
              << "indirect value: " << *pn << endl
              << "doubly indirect value: " << **ppn << endl
              << "address of n: " << pn << endl
              << "address of n via indirection: " << *ppn << endl;
     }
